# GeoProp Project Page

Static project page for **GeoProp: Grounding Robot State in Vision for Generalist Manipulation**.

## Preview

Open `index.html` directly in a browser, or run a local server:

```powershell
cd D:\Work\GeoProp\project_page
python -m http.server 8067
```

Then visit:

```text
http://127.0.0.1:8067/
```

## Files

- `index.html`: page structure and content.
- `styles.css`: visual system and responsive layout.
- `script.js`: small BibTeX copy interaction.
- `assets/`: paper PDF, compressed web images, and original source images.

The page is dependency-free and can be deployed directly to GitHub Pages or any static host.
